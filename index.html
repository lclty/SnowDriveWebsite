🚀 快速开始

Docker 一键部署

从 Docker Hub 拉取镜像，一分钟内启动你的私有云盘。

Terminal 
# 拉取镜像
$ docker pull lclty/snowdrive:latest

# 运行容器
$ docker run -d \
  --name snowdrive \
  -p 8080:8080 \
  -v /your/data/path:/data \
  -v /your/userdata/path:/userdata \
  -e SNOWDRIVE_SECRET_KEY=your-random-secret-string \
  lclty/snowdrive:latest

💡 中国大陆用户：如 Docker Hub 拉取缓慢，可从 Releases 页面下载镜像离线导入。
